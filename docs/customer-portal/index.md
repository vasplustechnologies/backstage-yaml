# Customer Portal

## Overview
The Customer Portal is the main web application used by customers to manage their accounts, view services, and make payments.

## Tech Stack
- React 18
- TypeScript
- Material-UI
- Redux Toolkit

## Key Features
- User authentication
- Dashboard with analytics
- Payment processing
- Ticket management

## Architecture
The Customer Portal is a single-page application that communicates with backend services through the Mobile API Gateway.

## Dependencies
- Mobile API Gateway
- User Service
- Redis Cache
