# Mobile API Gateway

## Overview
Serves as the single entry point for all client applications, routing requests to appropriate backend services.

## Tech Stack
- Node.js
- GraphQL
- Apollo Server
- Redis

## Features
- Request routing
- Authentication validation
- Rate limiting
- Response caching
- GraphQL federation

## GraphQL Schema
The gateway provides a unified GraphQL API that aggregates all backend services.

## Dependencies
- User Service
- Payment Service
- Inventory Service
- Notification Service
- Redis Cache
